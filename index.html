Padrão Observer

Nesse blog foi feita a aplicação do padrão de projeto Observer para criação de um projeto em que um ar condicionado digital é responsável por controlar a temperatura de uma sala de aula. Para isso o ar condicionado possui 6 sensores, três internos para calcular a quantidade de pessoas na sala, 3 três externos para calcular a temperatura externa.

O padrão observer é ideal nesse caso, pois permite que qualquer mudança percebida pelos sensores seja informada ao ar condicionado assim que ocorrer. Em nosso caso, utilizamos as classe próprias do java para isso, a classe Observable e a interface Observer.

Os métodos setChanged() e notifyObserver() são os responsáveis por informar a classe observadora (no caso a classe ArCondicionado) sobre qualquer mudança feita.
